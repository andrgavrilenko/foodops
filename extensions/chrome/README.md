# FoodOps Chrome Extension

> **Source of Truth** for all Chrome Extension implementation details.

## Overview

Automates adding grocery items to the shopping cart on S-kaupat.fi (S-Market) based on the user's weekly meal plan generated by FoodOps.

## Architecture

- **Manifest Version:** V3
- **Language:** TypeScript
- **Build Tool:** TBD (Vite + crxjs/vite-plugin recommended)
- **UI Framework:** React + TypeScript (popup)
